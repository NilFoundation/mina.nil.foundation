Detailed Description

template<std::size_t KeyBits>
class nil::crypto3::block::serpent< KeyBits >

Serpent. The most conservative of the AES finalists https://www.cl.cam.ac.uk/~rja14/serpent.html. An AES contender. Widely considered the most conservative design. Fairly slow, especially if no SIMD instruction set is available.
