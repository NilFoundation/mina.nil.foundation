Detailed Description

template<std::size_t BlockBits, std::size_t KeyBits>
class nil::crypto3::block::cast< BlockBits, KeyBits >

Cast-family ciphers. Variants available: Cast128, Cast256.

Cast128. A 64-bit cipher, commonly used in OpenPGP.

Cast256. A 128-bit cipher that was a contestent in the NIST AES competition. Rarely used, and now probably would be deprecated in crypto3. Use AES or Serpent instead.

Template Parameters
BlockBitsBlock cipher block bits. Does not represent the actual block bits value. Actual block bits value is BlockBits / 2. Available values are: 128, 256.
KeyBitsBlock cipher key bits.
