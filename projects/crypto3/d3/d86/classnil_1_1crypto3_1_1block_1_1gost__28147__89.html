Detailed Description

template<typename ParamsType = cbr_params>
class nil::crypto3::block::gost_28147_89< ParamsType >

GOST-28147-89 (Magma). 64-bit Russian cipher. Possible security issues. Avoid unless compatibility is needed.

Template Parameters
ParamsTypeGOST 28147-89 block cipher uses a set of 4 bit Sboxes, however the standard does not actually define these Sboxes; they are considered a local configuration issue. Several different sets are used.
