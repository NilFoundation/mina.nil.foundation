Detailed Description

template<std::size_t KeyBits>
class nil::crypto3::block::threefish< KeyBits >

Threefish. A variable-key length (512-bit is recommended) tweakable block cipher used in the Skein hash function via Merkle-Damg√•rd construction. Very fast on 64-bit processors.

Encrypt implemented directly from the Skein standard as found at http://www.skein-hash.info/sites/default/files/skein1.2.pdf

Template Parameters
KeyBitsBlock cipher key bits. Available values are: 256, 512.
