Detailed Description

template<typename Params, typename IV, typename Compressor, typename Finalizer = nop_finalizer>
class nil::crypto3::hash::haifa_construction< Params, IV, Compressor, Finalizer >

Template Parameters
DigestEndian
DigestBits
IV
Compressor
FinalizerThe HAIFA construction builds a block hash from a one-way compressor. As this version operated on the block level, it doesn't contain any padding or other strengthening. For a Wide Pipe construction, use a digest that will truncate the internal state.
Note
https://eprint.iacr.org/2007/278.pdf
